# TaskManager

A simple console-based **Task Manager** built in C++ using **object-oriented programming (OOP)** and **vectors**.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

## How to Run
1. Compile with g++:

```
g++ main.cpp Task.cpp -o taskmanager.exe

```
2. Run the program:

```
./taskmanager.exe

```
3. Follow the menu to manage tasks
